# Assets folder

Put your Encalm logo file here, named exactly:

    encalm_logo.png

(PNG with a transparent background works best. .jpg also works — if you use
a different format or filename, update the `LOGO_PATH` line near the top of
`Home.py` to match.)

Once the file is here, it will automatically appear at the top of the Home
page — no other code changes needed.
